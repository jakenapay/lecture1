Media Query

For example: @media (min-width: 700px) { background-color: red;}. This CSS code will tell that if the screen is 700px and wider, the background color should be red.



Another example: @media (max-width: 699px) { background-color: blue;}. This CSS code will tell that if the screen is 699px and smaller, the background color should be blue.
